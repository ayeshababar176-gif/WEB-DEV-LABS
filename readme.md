# Web Development Lab 2: CSS Grid

This lab focuses on learning the fundamentals of **CSS Grid** and how to create structured, responsive web layouts using modern CSS techniques.

## What I Learned

* `display: grid` — Converts an element into a grid container.
* `grid-template-columns` — Defines the number and width of grid columns.
* `grid-template-rows` — Defines the height and number of grid rows.
* `gap` — Adds spacing between grid items.
* `fr` units — Creates flexible and responsive layouts by distributing available space proportionally.

## Files

* `grid.html` — Practice file demonstrating the implementation of CSS Grid concepts and layout design.
