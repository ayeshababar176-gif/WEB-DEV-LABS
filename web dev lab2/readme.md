# Web Dev Lab 2: CSS Grid

In this lab, I learned how to build structured layouts using CSS Grid.

## What I Learned
* `display: grid` - Turns an element into a grid container
* `grid-template-columns` - Defines grid columns and sizes
* `grid-template-rows` - Defines grid rows
* `gap` - Adds space between grid cells
* Using `fr` units for flexible, responsive designs
